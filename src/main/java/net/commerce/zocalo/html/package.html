Support for generating HTML. {@link net.commerce.zocalo.html.HtmlElement} is the base class for {@link net.commerce.zocalo.html.HtmlSimpleElement}, which makes it easy to generate many simple HTML fragments, and {@link net.commerce.zocalo.html.HtmlRow}, which provides support for generating table rows given arrays of cells.

{@link net.commerce.zocalo.NumberDisplay} formats floating point numbers for printing.
