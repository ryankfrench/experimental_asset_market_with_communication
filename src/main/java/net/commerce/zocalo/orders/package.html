These classes describe market orders stored (in a book) for later use. {@link net.commerce.zocalo.orders.Order Orders} represent an order, while {@link net.commerce.zocalo.orders.SortedOrders} represents a collection of Orders on a single position that can find the best order and report on the best price and quantity available at that price. {@link net.commerce.zocalo.orders.OrdersIterator} iterates orders by price.

{@link net.commerce.zocalo.JspSupport.printOrders.OrdersPrintStrategy} is the base class for three classes that provide different strategies for printing orders interchangeably: {@link net.commerce.zocalo.JspSupport.printOrders.PQOrdersPrintStrategy}, {@link net.commerce.zocalo.JspSupport.printOrders.PriceOnlyOrdersPrintStrategy}, and {@link net.commerce.zocalo.JspSupport.printOrders.NoCancelOrdersPrintStrategy}.
